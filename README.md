# React Boilerplate

Created using [create-react-app](https://github.com/facebook/create-react-app), with some useful features.

## Features

- Ant Design
- Eslint
- Normalize.scss
- React
- React Router
- Redux
- Redux Thunk
- Scss
- Stylelint

## Basic Structure

```
src
|- components
|  |- App
|  |  |- App.js
|  |  |- AppActions.js
|  |  |- AppReducer.js
|  |  |- index.js
|  |- ...More Components
|- scenes
|  |- Home
|  |  |- Home.js
|  |  |- Home.scss
|  |- ...More Scenes
|- index.js
|- index.scss
|- routes.js
```

## Start

```bash
yarn && yarn start
```
